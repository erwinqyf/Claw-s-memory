## 🎯 推荐方案（3 选 1）

### 方案 A：通义万相网页版 ⭐ 推荐

**优点**: 免费、中文友好、生成质量高、无需配置

**步骤**:
1. 访问 https://tongyi.aliyun.com/wanxiang
2. 用支付宝/淘宝账号登录（免费）
3. 选择「文生图」功能
4. 复制下方提示词生成

**免费额度**: 每日约 20-50 次生成

**提示词（复制使用）**:

```
版本 A - 对比图：
专业社交媒体封面图，竖屏 3:4 比例。左侧绿色上升箭头和文字"+18%"加微笑表情，右侧红色下降箭头和文字"-12%"加沮丧表情，中间用 AI 机器人图标分隔。深灰色渐变背景。顶部大字"AI 翻译生产力 +40%"，底部小字"不会用=少赚 12%"。科技蓝和警示橙作为强调色。干净现代的数据可视化风格，高清专业。

版本 B - 极简风：
极简主义社交媒体封面，竖屏 3:4 比例。单个 AI 机器人手持翻译文档，背景放射光线。纯黑色背景。顶部霓虹蓝色大字"AI 翻译生产力"，中央超大"+40%"使用霓虹蓝和荧光绿渐变。底部现代无衬线字体"不会用=少赚 12%"。赛博朋克美学，干净大胆，高清专业。

版本 C - 数据图表风：
数据图表风格社交媒体封面，竖屏 3:4 比例。柱状图对比，左侧高柱 18% 蓝绿渐变，右侧矮柱 -12% 红橙渐变。白色到浅灰渐变背景。顶部文字"2025 译员收入报告"，图表上方"你会用 AI 吗？"，底部"现在学习还不晚"。专业商务图表风格，干净排版，现代企业美学，高清专业。
```

**尺寸设置**: 选择 3:4 或自定义 1242x1660

### 方案 B：Canva / 稿定设计 ⭐ 最快

**优点**: 模板丰富、可精确控制文字、立刻能用

**步骤**:
1. 访问 https://www.canva.com/ 或 https://www.gaoding.com/
2. 搜索「小红书封面」或「社交媒体封面」
3. 选择竖屏 3:4 模板
4. 按下方设计方案修改

**版本 A 设计要点**:
- 搜索「对比」「数据」模板
- 左右分栏布局
- 左侧绿色 +18%，右侧红色 -12%
- 中间加 AI/机器人元素
- 深色背景 + 亮色文字

**版本 B 设计要点**:
- 搜索「科技」「赛博朋克」模板
- 黑色背景
- 霓虹色文字（蓝/绿）
- 中央超大数字 +40%

**版本 C 设计要点**:
- 搜索「图表」「数据报告」模板
- 柱状图对比
- 商务风格
- 蓝绿 vs 红橙配色

---

### 方案 C：申请 Gemini API Key（自动化）

**优点**: 可批量生成、以后每期自动做

**步骤**:
1. 访问 https://aistudio.google.com/app/apikey
2. 用 Google 账号登录
3. 创建 API key
4. 告诉我 key，我配置后批量生成

**免费额度**: 每月约 60 次生成

**适合**: 长期运营、需要批量生产

## 🎨 3 个设计方案摘要

### 版本 A：对比图（推荐）
- **视觉**: 左右分割，+18% vs -12%
- **配色**: 深灰背景 + 科技蓝 + 警示橙
- **风格**: 数据可视化，专业感强
- **适合**: 突出数据冲击

### 版本 B：极简风
- **视觉**: AI 机器人 + 超大 +40%
- **配色**: 纯黑 + 霓虹蓝绿
- **风格**: 赛博朋克，年轻感
- **适合**: 吸引年轻受众

### 版本 C：数据图表
- **视觉**: 柱状图对比
- **配色**: 白灰背景 + 蓝绿/红橙
- **风格**: 商务报告，可信度高
- **适合**: 专业受众
