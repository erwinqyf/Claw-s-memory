**原因**:

2. Bravo 分析时未质疑来源真实性

4. Delta 发布前未做最终验证

**后果**: 所有内容基于虚假资讯，无法发布

## ✅ 新增强制质量关卡

### 关卡 0：来源验证（Bravo 负责）

**触发时机**: Echo 完成资讯搜集后

**检查项**:
- [ ] 每条资讯的链接是否可访问（HTTP 200）
- [ ] 来源是否权威（行业媒体、官方公告等）
- [ ] 日期是否在本周范围内
- [ ] 数据是否有具体细节（非模糊描述）

**工具**:
```bash
# 批量检查链接
python3 skills/content-verification/scripts/verify-links.py <文件>

# 手动抽查
curl -sI "URL" | head -1
```

**通过标准**:
- ✅ 所有链接 HTTP 200
- ✅ 来源为白名单媒体（Slator、ProZ 等）
- ✅ 日期在 7 天范围内
- ✅ 数据有具体数字和来源

**不通过处理**:
- ❌ 有任何链接失效 → 退回 Echo 重新搜集
- ❌ 来源可疑 → 标记并人工审核
- ❌ 日期超范围 → 标注"非本周新闻"

---

### 关卡 1：独立审查（Charlie 负责）

**触发时机**: Bravo 完成分析后

**检查项**:
- [ ] 随机抽取 2-3 条资讯，重新搜索验证
- [ ] 检查核心数据是否有多个来源支撑
- [ ] 识别"幻觉信号"（过于完美的数据、单一来源）
- [ ] 给出可信度评分

**可信度评分**:

| 等级 | 标准 | 处理 |
|------|------|------|
| **高** | 多来源交叉验证，无红旗 | ✅ 进入下一工序 |
| **中** | 单一权威来源，无明显问题 | ⚠️ 标注"单一来源"后继续 |
| **低** | 来源不明、有红旗信号 | ❌ 退回重新搜集 |

**红旗信号清单**:
- ⚠️ 单一来源，无其他媒体报道
- ⚠️ 数据过于精确（如：1.42 亿英镑，恰好 23% 增长）
- ⚠️ 无法找到原始报告/研究
- ⚠️ 引用"匿名消息人士"
- ⚠️ 链接是示例格式（example.com、domain.com/test）
- ⚠️ 内容与常识不符

---

### 关卡 2：最终验证（Delta 负责）

**触发时机**: 内容创作完成后、发布前

**检查项**:
- [ ] 文案中的数据与原始资讯一致（未夸大/曲解）
- [ ] 所有链接再次验证（防止发布前失效）
- [ ] 标题未过度夸张（与内容匹配）
- [ ] 标签准确（未蹭无关热点）

**最终检查清单**:
```markdown
## 发布前检查清单

- [ ] 所有资讯链接已验证（HTTP 200）
- [ ] 来源权威性已确认（白名单媒体）
- [ ] 数据已交叉验证（非单一来源）
- [ ] 无"幻觉信号"（过于完美的数据）
- [ ] 文案未夸大/曲解原始资讯
- [ ] 标题与内容匹配（非标题党）
- [ ] 标签准确（未蹭无关热点）
- [ ] 可信度评分：高/中

**审查人**: [姓名]
**审查时间**: YYYY-MM-DD HH:MM
**是否批准发布**: ✅ 是 / ❌ 否
```

## 🎯 执行流程示例

### 正常流程

```
1. Echo 搜集 8 条资讯
   ↓
2. Bravo 验证链接（关卡 0）
   - 8 条全部 HTTP 200 ✅
   - 来源均为 Slator ✅
   - 日期在本周内 ✅
   ↓
3. Bravo 分析趋势
   ↓
4. Charlie 独立审查（关卡 1）
   - 随机抽查 3 条，重新搜索验证 ✅
   - 无红旗信号 ✅
   - 可信度：高 ✅
   ↓
5. Charlie 创作文案
   ↓
6. Delta 最终验证（关卡 2）
   - 数据与原文一致 ✅
   - 链接再次验证 ✅
   - 标题未夸张 ✅
   ↓
7. 发布 ✅
```

### 异常流程（发现问题）

```
1. Echo 搜集 8 条资讯
   ↓
2. Bravo 验证链接（关卡 0）
   - 3 条链接失效 ❌
   - 2 条来源可疑 ⚠️
   ↓
3. 触发异常处理
   - 退回 Echo 重新搜集
   - 记录失败原因
   - 通知人工审核
   ↓
4. Echo 重新搜集（仅保留 5 条真实资讯）
   ↓
5. 继续流程...
```

## 🫡 原则重申

**宁可说"本周无重大新闻"，也不要编造虚假资讯！**

**发布节奏**:
- 有真实新闻 → 按流程发布
- 无真实新闻 → 改做教程/洞察类内容
- 资讯不足 → 延期发布，不凑数

**质量优先于数量，真实性优先于时效性。**
