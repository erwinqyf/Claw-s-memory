## 🎯 格式排版说明

### 1. 标题层级

| 层级 | 格式 | 示例 |
|------|------|------|
| 论文标题 | 14磅粗体居中 | 不同语言模型在文学翻译任务上的评估效果对比研究 |
| 一级标题 | 12磅粗体 | 1 引言 |
| 二级标题 | 10磅粗体 | 1.1 研究背景 |
| 三级标题 | 10磅粗体嵌入式 | Run-in Heading in Bold. |
| 四级标题 | 10磅斜体嵌入式 | *Lowest Level Heading.* |

### 2. 正文格式

**摘要格式**:
```
背景: ...
方法: ...
结果: ...
结论: ...

关键词: 文学翻译;机器翻译评估;...
```

**正文段落**:
- 首行缩进：除章节首段和图表后首段外，其他段落首行缩进
- 行距：单倍行距
- 字体：宋体（中文）/ Times New Roman（英文）

### 3. 表格格式

**表格标题**：置于表格上方
```
表 1. 段落级评估结果

| 排名 | 模型配置 | BLEU | ... |
|------|----------|------|-----|
| ... | ... | ... | ... |
```

### 4. 公式格式

**公式居中，单独成行**：
```
    x + y = z                                            (1)
```

### 5. 图片格式

**图片标题**：置于图片下方
```
[图片]

图 1. 图片标题说明
```

### 6. 引用格式

**参考文献**：使用方括号编号 [1], [2]...
```
文学翻译长期以来被视为机器翻译的"最后堡垒"[1]。
```

## 📝 使用 Word 模板的步骤

### 步骤 1：打开模板
1. 打开 `waica2026.docm` 模板文件
2. 启用宏（如果需要）

### 步骤 2：复制内容
从 `waica2026_chinese_formatted.txt` 复制内容到 Word

### 步骤 3：应用样式
使用 Word 样式面板应用对应样式：

| 内容 | 应用样式 |
|------|---------|
| 论文标题 | papertitle |
| 作者姓名 | author |
| 单位地址 | address |
| 摘要 | abstract |
| 关键词 | keywords |
| 一级标题 | heading1 |
| 二级标题 | heading2 |
| 三级标题 | heading3 |
| 四级标题 | heading4 |
| 正文 | p1a |

### 步骤 4：调整表格
1. 复制表格内容
2. 在 Word 中插入表格
3. 应用 tablecaption 样式（表格标题在上方）

### 步骤 5：调整公式
1. 使用 Word 公式编辑器重新输入公式
2. 公式居中，编号右对齐

### 步骤 6：调整图片
1. 插入图片
2. 应用 figurecaption 样式（图片标题在下方）

## ⚠️ 注意事项

### 需要手动调整的部分

1. **引用格式转换**
   - 原文：(Toral & Way, 2018)
   - WAICA：[1]
   - 需要手动编号并创建参考文献列表

2. **公式转换**
   - 原文：LaTeX 格式
   - WAICA：Word 公式编辑器
   - 需要重新输入

3. **表格转换**
   - 原文：Markdown 表格
   - WAICA：Word 表格
   - 需要重新创建

4. **图片**
   - 原文：无图片
   - 如需添加，使用 figurecaption 样式
