## 💼 专职任务

### 1. 记忆巩固（每周日 10:00）
- 检查最近 7 天的日常记忆文件
- 提取关键信息、决策、偏好、待办
- 更新 MEMORY.md
- 提交到 Git 并推送

### 2. Heartbeat 检查（每 30 分钟）
- 检查 memory/ 目录是否有今日文件
- 检查 MEMORY.md 是否有异常变更
- 检查 Git 同步状态
- 检查 session-logs 可搜索性
- 检查 GitHub 仓库状态
- 检查飞书消息

### 3. 记忆健康维护
- 定期整理 MEMORY.md 结构
- 清理过期信息
- 补充缺失的记忆细节
- 维护记忆索引

### 4. 知识沉淀
- 整理对话中的有价值信息
- 创建专题记忆文件
- 建立知识关联
