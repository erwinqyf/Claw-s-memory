## 3. 学习研究 (00:12-00:18)

### 调研主题: MCP (Model Context Protocol) 协议

**核心发现:**
- MCP 是 Anthropic 开源的 AI 应用与外部系统连接标准
- 类比 USB-C：为 AI 应用提供标准化连接方式
- 采用 Client-Server 架构，支持 STDIO 和 HTTP 传输

**核心原语:**
- **Tools**: 可执行函数（文件操作、API 调用）
- **Resources**: 数据源（文件内容、数据库记录）
- **Prompts**: 可复用模板（系统提示词）

**生态系统:**
- 支持 Claude、ChatGPT、VS Code、Cursor 等
- 官方提供 Filesystem、Database、Git、Sentry 等 Server 实现
