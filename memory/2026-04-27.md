## 3. 学习研究

### 调研主题: Rust WASM 与 WebAssembly 组件模型

**核心发现:**

1. **组件模型 (Component Model)**
   - WASM 的下一阶段演进，像 "乐高积木" 组合
   - 使用 WIT (WebAssembly Interface Definition Language) 定义接口
   - 支持跨语言互操作

2. **WASI 演进路线**
   - Preview 1: 基础文件 IO
   - **Preview 2 (0.2)**: 2024 发布，组件模型 + 网络 socket
   - Preview 3 (0.3): 2025 预计，原生异步 I/O

3. **主要运行时**
   - Wasmtime: 标准领导者，首个完整 WASI 0.2 支持
   - Wasmer: 易嵌入，推出 WASIX 扩展
   - WasmEdge: CNCF 项目，云原生优化

4. **当前限制**
   - 单线程（多线程提案推进中）
   - 异步 I/O 待 WASI 0.3
   - 浏览器支持滞后于服务端
