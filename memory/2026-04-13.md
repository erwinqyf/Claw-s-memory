**调研主题：** OpenClaw ACP (Agent Client Protocol)

**核心发现：**
1. ACP 是连接 IDE 与 OpenClaw Gateway 的桥梁协议
2. 通过 stdio 通信，支持 Codex、Claude Code、Gemini CLI 等外部工具
