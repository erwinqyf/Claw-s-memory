### 学习研究详情

**调研主题:** AI Agent Memory 架构 (2024-2025)

**核心发现:**
1. **三大核心架构对比**
   - RAG: 简单、低成本、可解释
   - Vector Stores: 语义搜索、海量数据、速度快
   - GraphRAG: 多跳推理、关系可解释、准确性高

2. **认知科学映射**
   - 感觉记忆 → 输入缓冲区
   - 工作记忆 → Context Window
   - 短期记忆 → In-memory cache
   - 长期记忆 → Vector DB / Graph DB

3. **技术选型建议**
   - 快速原型: Vector Store
   - 生产级语义搜索: Vector + RAG
   - 复杂关系推理: GraphRAG
   - 混合场景: Vector + Graph 混合
