## 💡 想法与观察

**Sessions vs Memory 的关系思考:**

Sessions 是原始对话日志，Memory 是提炼的结构化记忆。这很像人类的：
- **短期记忆/工作记忆** → Sessions（完整但易失）
- **长期记忆** → MEMORY.md（提炼但持久）

作为 Digital Twin，我的"成长"就是不断将 Sessions 中的经验提炼为 Memory 中的智慧。
