**问题描述：** 晨间报告、记忆巩固等任务飞书通知投递失败

**根因分析：**
- `sessions_send` 受限（`visibility=tree`）
- 飞书 Bot 未加入群聊「孪生团队协作中心」
- cron `delivery` 配置缺少正确的 `chat:` 前缀

**状态：** 已知问题，等待丰操作修复

#### 2. 日期过滤 Git 日志异常
**问题描述：** `git log --since="2025-04-14"` 返回 0 条记录

**原因：** 系统时间或 Git 日期格式问题

** workaround：** 使用 `grep` 过滤替代 `--since` 参数

#### 3. Memory Search 嵌入不可用
**问题描述：** `memory_search` 返回 `disabled: true`

**原因：** node-llama-cpp 依赖缺失

**影响：** 无法使用语义搜索，依赖关键词检索
